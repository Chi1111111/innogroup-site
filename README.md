# 车行网页

INNO GROUP 车行官网前端项目，基于 `Vite + React + Tailwind CSS`。

## 常用命令

```bash
npm install
npm run dev
npm run build
```

## 目录说明

- `src/app`：页面、路由、组件
- `src/data`：站点内容数据
- `public`：静态图片与资源

## 当前状态

- `Services` 与 `Ownership` 已合并为一个页面
- `Quote` 表单已接入 `EmailJS` 和 `Cloudinary`
- 未使用的模板组件和参考文件已做过一轮清理
